Шаг 1: Скачать установочный файл Steam
Скачать Steam
Шаг 2 Создать текстовый файл и спрятать его в папке, где вы сможете его найти, но его не увидит учитель
Совет: Текстовый файл и установочный файл Steam лучше хранить в одной папке
Шаг 3: Вписать следующий текст:
set __COMPAT_LAYER=RunAsInvoker && start "" %1
Шаг 4: Переименовать файл в "*любое_название*.bat"
Шаг 5: Перетащить установочный файл Steam на файл "*любое_название*.bat"
Совет: Steam не запустится при установке на диск C или в директорию с Кириллицей в пути, нужно создать папку с названием на английском на диске D и установить Steam в неё
Шаг 6: Ждите окончания установки
Шаг 7: Теперь установите GoodbyeDPI, он не заблокирован, смело гуглите

Шаг 7а: Перейдите на официальный Github GoodbyeDPI


Шаг 7б: В параграфе Quick Start перейдите по ссылке "For Russia: Download latest version from Releases page"


Шаг 7в: Во вкладке Assets выбрать "goodbyedpi-0.2.3rc3-2.zip"


Шаг 7г: Распакуйте архив


Шаг 7д: Откройте файл russia-blacklist.txt и вставьте туда ссылку store.steampowered.com


Совет: Это разблокирует ТОЛЬКО магазин Steam, для полноценной работоспособности Steam нужно добавить в russia-blacklist.txt остальные ссылки


Шаг 7е: Запустите файл 1_russia_blacklist.cmd так же, как запускали Steam (повторить шаг пятый)


Шаг 8: Запустить Steam и наслаждаться)
Совет: Для того, чтобы учитель не спалил контору, в настройках Steam или в диспетчере задач отключите автозапуск Steam
