Шаг первый: Скачать установочный файл Steam
Скачать Steam
Шаг второй: Создать текстовый файл и спрятать его в папке, где вы сможете его найти, но его не увидит учитель
Совет: Текстовый файл и установочный файл Steam лучше хранить в одной папке
Шаг третий: Вписать следующий текст:
set __COMPAT_LAYER=RunAsInvoker && start "" %1
Шаг четвёртый: Переименовать файл в "*любое_название*.bat"
Шаг пятый: Перетащить установочный файл Steam на файл "*любое_название*.bat"
Совет: Steam не запустится при установке на диск C или в директорию с Кириллицей в пути, нужно создать папку с названием на английском на диске D и установить Steam в неё
Шаг шестой: Ждите окончания установки
Шаг седьмой: Теперь установите GoodbyeDPI, он не заблокирован, смело гуглите
Шаг 7а: Перейдите на официальный Github GoodbyeDPI
Шаг 7б: В параграфе Quick Start перейдите по ссылке "For Russia: Download latest version from Releases page"
Шаг 7в: Во вкладке Assets выбрать "goodbyedpi-0.2.3rc3-2.zip" Шаг 7г: Распакуйте архив Шаг 7д: Откройте файл russia-blacklist.txt и вставьте туда ссылку Шаг девятый: Запустить Steam и наслаждаться) Совет: Для того, чтобы учитель не спалил контору, в настройках Steam или в диспетчере задач отключите автозапуск Steam
